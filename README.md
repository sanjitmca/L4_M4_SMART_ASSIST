# SmartAssist
How to Run the Console Application

1.Open the solution in Visual Studio (SmartAssist.sln).

2.In Solution Explorer, right‑click on SmartAssist.ConsoleApp → Set as Startup Project.

3.Press F5 (Run) or Ctrl+F5 (Run without debugging).

4.The console window will open and show output.
